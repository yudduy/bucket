# Bucket

A goal-tracking social app for iOS 17. Create bucket list goals, post progress updates, and share your journey with friends.

## Features

- Create and track bucket list goals
- Post progress updates with photos
- Follow friends and engage with their progress
- Real-time notifications

## Tech Stack

- SwiftUI + Combine
- Firebase (Auth, Firestore, Storage)
- MVVM + Clean Architecture
- Dependency injection with Factory
